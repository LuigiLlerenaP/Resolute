# Resolute

Resolute is a small .NET library for modelling Result and Fault types to simplify error handling and functional-style flows.

Installation (after publishing):

	dotnet add package Resolute --version 1.0.0

Quick usage

- See the Results and Failures folders for Result, Fault and AggregateFault types.

Building and packing locally

- To build: `dotnet build -c Release`
- To create a NuGet package locally: `dotnet pack -c Release`
